# Symple-3DRPG_Practice
３Dでゲームを作成する練習をしているため、そのログを取っています。もし使えるものがあればご自由にどうぞ。

## ソースコード
ソースコードはBrunchのmasterにおいてあります。そちらをご覧ください。

## 現在実装している内容(2025/12/22)
- プレイヤー移動
- カメラ追従・回転
- 攻撃判定
- ダメージ処理
- HPバー表示
- ノックバック
- Enemy AI（ランダム移動 / 追跡）
- Terrainで地面作成
- Animatorでアニメーションを挿入(Playerのみ)

##　進捗と問題(2025/12/22)
- Playerにanimationを実装しました。
- Playerのモデルを（Mixamoでインストールして）実装しました。
- アニメーションをCharacter Animationで作成しました。
- Animator は Player 本体ではなく Model に配置
- 攻撃は「アニメーション再生」と「当たり判定」を分離
- 攻撃判定は Coroutine / Animation Event で制御

## 進捗と問題(20225/12/19)
- ノックバックがきちんと反応しない問題を解消しました。
- Enemyの攻撃判定が複数回呼ばれ、クールダウン時間を無視して攻撃する問題を解消しました。
- Enemyの攻撃が一度攻撃後に再度攻撃できなくなる問題が発生したため、解消に向けて問題の原因を精査しています。
- 地面のブロックを、ちゃんとした地形とマテリアルに変更しました。
- プレイ可能エリアを設定しました。

## 進捗と問題(2025/12/18)

- Enamy側の攻撃クールダウンに異常が発生。予備動作時に攻撃判定が出てめっちゃ攻撃を食らう。
- Hpバーを追加しました。
- Hpバーが攻撃を受けた時（Hp減少時）に揺れたり赤色になって減少が見やすくしました。

 その後すぐに修正しました。
